# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-01-XX

### Adicionado
- Sistema RAG (Retrieval Augmented Generation) para busca semântica
- Botão de indexação manual do workspace
- Suporte para múltiplos modelos de IA
- Automação de navegador integrada
- Sistema de checkpoint de mudanças

### Modificado
- Nome do projeto alterado de Void para PolliDev
- Atualização de informações do autor
- Melhorias na interface do usuário
