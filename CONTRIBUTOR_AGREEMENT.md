# Acordo de Contribuidor - PolliDev

Ao contribuir para o projeto PolliDev, você concorda com os seguintes termos:

## 1. Transferência de Direitos

Ao enviar uma contribuição (Pull Request, Issue, ou qualquer outra forma de contribuição), você automaticamente transfere todos os direitos autorais e propriedade intelectual de sua contribuição para **Fabio Arieira Baia**, proprietário do projeto PolliDev.

## 2. Restrições de Uso

Você concorda que:

- ✅ **PODE**: Contribuir com melhorias, correções e features para o projeto PolliDev
- ✅ **PODE**: Usar o software para fins pessoais ou comerciais
- ✅ **PODE**: Reportar bugs e sugerir melhorias
- ❌ **NÃO PODE**: Copiar o código para criar projetos derivados
- ❌ **NÃO PODE**: Usar o código em produtos concorrentes ou similares
- ❌ **NÃO PODE**: Remover ou alterar avisos de direitos autorais
- ❌ **NÃO PODE**: Vender, licenciar ou distribuir o código como seu próprio produto
- ❌ **NÃO PODE**: Criar forks com intenção de criar um produto separado

## 3. Propriedade Intelectual

Todas as contribuições tornam-se propriedade exclusiva de **Fabio Arieira Baia**. Você não retém direitos sobre o código que contribuiu, exceto o direito de continuar usando o software conforme os termos de licença.

## 4. Aceitação

Ao fazer um Pull Request, criar uma Issue, ou de qualquer outra forma contribuir para o projeto, você confirma que:

1. Leu e entendeu este acordo
2. Concorda com todos os termos aqui estabelecidos
3. Tem autoridade legal para aceitar estes termos
4. Suas contribuições não violam direitos de terceiros

## 5. Consequências de Violação

Qualquer violação destes termos resultará em:

- Rejeição imediata de suas contribuições
- Banimento permanente do repositório
- Possível ação legal por violação de direitos autorais
